# Slowoku Bot

Python bot created for automating interactions with the Slowoku word puzzle game.  
The project focuses on text processing, program logic design, and building structured Python applications.

## Description

Slowoku Bot is a simple Python-based program that processes text input and generates automated responses for the Slowoku game.  
The goal of the project was to practice working with Python, text handling, and building a clean program structure.

This project was created as part of learning Artificial Intelligence and Data Science tools.

## Technologies

- Python
- NLP / text processing
- Standard Python libraries
- Git / GitHub

## Features

- Text input processing
- Automated response generation
- Modular code structure
- Easy to modify and extend

The project was created to:

- build structured applications
- gain experience with GitHub projects
